# 💰 Personal Finance Dashboard (Tableau)

An interactive Tableau dashboard built to analyze personal income and expense patterns using a sanitized financial dataset. This project demonstrates data cleaning, transformation, and visualization skills for financial analysis.

---

## 📊 Dashboard Overview

This dashboard provides a comprehensive view of financial activity through:

* **KPI Cards** for total income and total expenses
* **Expense Distribution** using a treemap visualization
* **Monthly Expense Trend** to identify spending patterns over time

---

## 🛠 Tools & Technologies

* Tableau (Data Visualization)
* Python (Data Cleaning & Preprocessing)
* CSV (Structured Dataset)

## 🔐 Data Privacy

The dataset used in this project is a **sanitized version of personal financial data**:

* Sensitive information (merchant names, IDs, source files) has been removed
* Values may be slightly modified to preserve privacy while maintaining realistic patterns

---

## 📈 Key Insights

* Housing and transfers account for the majority of expenses
* Spending shows noticeable spikes in specific months
* Overall income exceeds expenses, indicating positive savings

---

## 🚀 How to Use

1. Download the `.twbx` file from the `dashboard/` folder
2. Open it using Tableau Desktop or Tableau Public
3. Explore the dashboard interactively
